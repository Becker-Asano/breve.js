    Overview

    The main breve simulation Engine which manages the simulation state.

    Instance Method Summary

    • - (void) click(location, target, event) Invoked on the engine when a click occurs in the rendered simulation area.
    • - (void) setup() Sets up the simulation with a set of provided attributes.
    • - (breve.Agent) add(agent) Bound Adds an agent to the simulation.
    • - (Array) all(agentType) Bound Returns an array of all agents in the simulation matching the given type.
    • - (void) start() Bound Starts the simulation, scheduling a timer to step the simulation forward in time 60 times per second until stop is called.
    • - (void) stop() Bound Stops the simulation and clears all objects.
    • - (Object) bounds() Gives the bounding box of the simulation rendering area.
    • - (void) step() Bound Steps the simulation forward in time by the timeStep value.
    • - (void) debug(msg) Sets a debug message to be displayed on the simulation page.
    • - (void) draw(canvasContext) Bound Draws the background of the simulation.
